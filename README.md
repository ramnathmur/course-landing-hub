# NEURAL HUB — Educational Intelligence for Practitioners

A cyberpunk-themed landing hub serving as the entry point for educational courses on AI, agents, and intelligent systems.

## Courses

1. **LLM Intelligence & Agent Autonomy** — From next-word prediction to autonomous action
2. **Agents Master Course** — Building, deploying, and managing AI agents
3. **book-to-html** — Interactive digital book readers
4. **AI-first App Dev Blueprint** — Practical patterns for AI-driven applications
